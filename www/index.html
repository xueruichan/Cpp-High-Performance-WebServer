Epoll EventLoop

底层采用 Linux Epoll I/O 多路复用技术与非阻塞 Socket，主线程极限专注于事件分发，轻松应对海量高并发连接。

⚙️

Thread Pool

自研基于条件变量与互斥锁的线程池，实现工作线程与网络层彻底解耦，榨干多核 CPU 计算性能。

🔀

Dynamic Router

面向对象的 HTTP 解析器配合 Lambda 闭包路由系统，支持静态文件回源与动态 RESTful API 的优雅挂载。

📝

Async Logger

独立的后台监控线程与阻塞队列，实现业务逻辑与磁盘 I/O 零冲突，精准捕捉每一条运行轨迹与异常告警。
